## 무엇인가

[galaga-clone](https://github.com/injoon5/galaga-clone)은 Python과 pygame으로 짠 갤러그 스타일 세로 슈팅 게임입니다. 파일 하나(`galaga.py`), 약 200줄, 스프라이트 PNG, 그리고 BGM 트랙이 함께 들어 있습니다. 당시 코딩 학원(C3 Coding)에서 게임 개발을 배우고 있었고, 이게 일종의 졸업 작품 같은 프로젝트였습니다.

재시작 로직을 손본 `beta/main.py`도 있고, 진행하면서 풀어 본 작은 pygame 연습 — 마우스 추적, 폰트 렌더링, BGM 재생 — 들로 가득한 `tutorials/` 폴더도 있습니다.

## 어떻게 플레이하나

640×650 창, 60 FPS. 화면 아래쪽의 우주선(`forces.png`)을 조종하는데, **좌우 방향키**로 이동하고(프레임당 3px, 화면 경계에 고정), **스페이스**로 발사합니다.

적(`enemy.png`)은 **1초마다** 화면 위쪽 임의의 X 위치에서 등장합니다. 각 `Enermy`(네, 코드에서 제가 이렇게 철자를 틀렸습니다)는 임의의 낙하 속도(2–6)와 수평 방향을 고른 뒤 **아래로 가속**합니다(매 프레임 `dy += 0.1`). 좌우 벽에 부딪히면 `dx`를 뒤집어 튕깁니다.

미사일은 스프라이트가 아니라 `pygame.draw.line`으로 그린 **너비 6px짜리 선**이고, 매 프레임마다 **임의의 RGB 색**을 입습니다. 아주 2020년스러운 감성이죠.

적 명중 → 점수 +1, 두 객체 모두 삭제. 적이 우주선에 도달 → 게임 오버 오버레이(`game-over.png`)와 함께 재시작 버튼 등장. 재시작을 누르면 다시 시작합니다.

## 코드 구조

루프 하나에 클래스 넷:

| 클래스 | 역할 |
| ----- | --- |
| `Enermy` | 등장, 이동, 튕김, 미사일과 충돌 |
| `Forces` | 플레이어 우주선 — 이동, 그리기, 발사, 적과 충돌 |
| `Missile` | 프레임당 5px로 위로 비행, 화면 밖이면 삭제 |
| `Button` | 재시작 이미지 히트 테스트 |

충돌 처리는 화려하게 수동입니다 — 중첩 `while` 루프로 인덱스를 줄여 가며 리스트에서 삭제하죠. pygame 스프라이트 그룹도, 물리 엔진도 없습니다. 오직 rect와 `collidepoint`, 그리고 희망으로 버팁니다.

게임 오버 상태는 `gaming = False`로 설정하고, 적을 모두 지우고, 점수를 0으로 리셋한 뒤, `(250, 510)`에 있는 재시작 버튼 rect 위에서의 마우스 클릭을 기다립니다.

## 에셋

- `enemy.png`, `forces.png` — 검은색 컬러키가 적용된 우주선 스프라이트
- `game-over.png`, `restart.png` — 종료 화면 UI
- `Forget Me Not - Patrick Patrikios.mp3` — 배경 음악 (튜토리얼 폴더에 pygame BGM 로딩 코드가 있습니다)

## 솔직한 평가

요즘 기준으로는 엉성합니다 — 클래스 이름의 오타, while 루프 안에서의 리스트 변형, 재시작을 누르기도 전에 게임 오버에서 점수가 리셋되는 것까지요. 하지만 처음부터 끝까지 완성한 첫 게임이었고, 적이 튕기는 물리를 작동시켰을 때의 그 뿌듯함은 아직도 기억납니다. README에는 그냥 `c3coding pygame`이라고만 적혀 있고, 결국 흐지부지된 후속작 `codingPro01/galaga`를 가리킵니다.
